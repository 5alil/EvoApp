Welcome to Evolvice Task

This is a simple project for showing Repository Design Pattern and use technologies :

1- ASP.Net Web API .Net Core 2.1
2- .Net Core Standared
3- .Net Entity Framework Core 2.1
4- SQL SERVER DB
5- Angular 5
6- Bootstrap

Front End Architecture and How to call End Points:

Angular Component --> Shared Service --> API End Point

Back End Architecture and How to hite the Database:

Controller --> Service --> Repository --> DB
